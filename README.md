# Toastr 
Toastr Alert anywhere.

## Display Toastr Alert
```sh
toastr.success('Toastr install successfully.')
toastr.info('Also use your Name.')
toastr.warning('Are you sure?')
toastr.danger('Are you sure?')
```

## Options 
```sh
toastr.options = {
  "closeButton": false,
  "debug": false,
  "newestOnTop": false,
  "progressBar": false,
  "positionClass": "toast-top-right",
  "preventDuplicates": false,
  "onclick": null,
  "showDuration": "300",
  "hideDuration": "1000",
  "timeOut": "5000",
  "extendedTimeOut": "1000",
  "showEasing": "swing",
  "hideEasing": "linear",
  "showMethod": "fadeIn",
  "hideMethod": "fadeOut"
}
```
